Telephus
========

Description
-----------

Telephus is a connection pooled, low-level client API for Cassandra in Twisted python.

Installation
------------

Prerequisites:

* Python >= 2.4 (2.5 or later for tests)
* Twisted 8.1.0 or later
* Thrift (latest svn)

Usage
-----

See example.py for an example of how to use the Telephus API.
